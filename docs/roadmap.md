# AI Agent System Roadmap

## 项目目标

构建一个企业级 AI Agent 系统，具备：

- RAG 知识库
- Agent 工作流
- Tool Calling
- 多轮记忆
- 文档解析
- Workflow 自动化
- 企业级 API
- Docker 部署

目标：

- 作为 AI 工程项目写入简历
- 用于实习 / 求职
- 后续可扩展接单
- 学习 AI Engineering
